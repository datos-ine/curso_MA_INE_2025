Meta-análisis para estudios descriptivos

Para estudios observacionales de tipo descriptivo, los estimadores de efecto más comunmente utilizados son las proporciones y las correlaciones.

Meta-análisis de prevalencia

La prevalencia de un evento de salud se calcula como una proporción usando la fórmula:

\[ p = \frac{k}{n} \]

donde:

  • \(k\) es el número de individuos con el evento.

  • \(n\) es el número total de individuos.

El error estándar de la proporción se estima como:

\[ SE_p = \sqrt{\frac{p(1-p)}{n}} \]

Debido a que las proporciones tienen un rango de entre 0 y 1, cuando nos acercamos a alguno de estos valores se comprime el error estándar y se produce la sobreestimación de los resultados.

Una forma común de solucionar el problema es aplicar la transformación logit a las proporciones crudas. La función escalc() del paquete metafor nos permite transformar las proporciones a logit usando los siguientes argumentos:

{r} #| eval: false prev <- escalc(measure = "PLO", # transformación logit de proporciones xi = evento, # individuos con el evento ni = n, # total de individuos en la muestra slab = titulo, # identificador del estudio data = datos) # base de datos de interés}

Veamos un ejemplo práctico usando la base de datos “dat.crisafulli2020” incluida en el paquete metadat. Comenzamos cargando la base de datos y haciendo una exploración rápida:

{r} # Carga datos datos <- dat.crisafulli2020 # Exploración datos glimpse(datos)}

Nuestras columnas de interés son study, cases y total, que contienen el identificador de estudio, el número de individuos con el evento y el número total de individuos muestreados en cada estudio. Aplicamos la transformación logit:

{r} # Crea nuevo dataset con los datos transformados prev <- escalc(measure = "PLO", # transformación logit de proporciones xi = cases, ni = total, slab = study, data = datos) # Explora objeto glimpse(prev)}

En el objeto que creamos con escalc(), aparecen dos nuevas columnas: yi y vi que corresponden al estimador de efecto y su varianza, respectivamente. Una vez aplicada la transformación, podemos correr el modelo de meta-análisis usando la función rma() de metafor:

{r} mod_prev <- rma(yi = yi, vi = vi, measure = "PLO", slab = study, data = prev)}

Una forma más sencilla de llegar al mismo resultado, es usando la función metaprop() del paquete meta, donde en un solo paso podemos transformar los datos y ajustar el modelo de meta-análisis usando los siguientes argumentos:

{r} mod_prev2 <- metaprop(event = cases, # individuos con el evento n = total, # total de individuos en la muestra sm = "PLOGIT", # transformación logit studlab = study, # identificador del estudio data = prev) # base de datos de interés}

Comparemos la salida de ambos modelos:

{r} summary(mod_prev) summary(mod_prev2)}

Meta-análisis de incidencia

Meta-análisis de correlaciones

Las correlaciones indican la fuerza y dirección de la covarianza entre dos variables numéricas continuas y se expresa como:

\[ r_{xy} = \frac{Cov_{xy}}{S_xS_y} \]

donde:

  • \(Cov_{xy}\) es la covarianza entre las variables X e Y.

  • \(S_x\) y \(S_y\) son los desvíos estándar para cada variable.

El error estándar para la correlación se puede expresar como:

\[ SE = \frac{1 - r^2_{xy}}{\sqrt{n-2}} \]

Las correlaciones pueden tomar valores de entre -1 y 1, por lo que se pueden introducir sesgos cuando calculamos el error estándar para muestras pequeñas. Para trabajar con meta-análisis de correlaciones, los datos se transforman usando el coeficiente z de Fisher, que aproxima la muestra a una distribución normal.

En el paquete metafor podemos realizar la transformación de los datos usando la función escalc() con los siguientes argumentos:

{r} #| eval: false cor <- escalc(measure = "ZCOR", # transformación de Fisher ri = correlacion, # vector de correlaciones ni = n, # total de individuos en la muestra slab = titulo, # identificador del estudio data = datos) # base de datos de interés}

La función metacor() del paquete meta, nos permite ajustar modelos de meta-análisis para proporciones simples sin necesidad de transformar previamente los datos usando el argumento sm = "ZCOR":

{r} #| eval: false mod_cor <- metacor(cor = correlacion, # vector de correlaciones n = n, # total de individuos en la muestra sm = "ZCOR", # transformación de Fisher studlab = titulo, # identificador del estudio data = datos) # base de datos de interés}

Meta-análisis de estudios analíticos

Cuando trabajamos con datos de estudios epidemiológicos analíticos, podemos realizar meta-análisis para distintas medidas de asociación.

Diferencia de medias

La diferencia de medias entre dos o más grupos se define como:

\[ MD = \bar{x_1} - \bar{x_2} \]

donde:

  • \(\bar{x_1}\) es la media muestral del grupo 1.

  • \(\bar{x_2}\) es la media muestral del grupo 2.

El error estándar para la diferencia de medias se obtiene como:

\[ SE_{md} = S_{comb} \sqrt{\frac{1}{n_1} + \frac{1}{n_2}} \]

donde \(n_1\) y \(n_2\) son los tamaños muestrales en ambos grupos y \(S_{comb}\) es el desvío estándar combinado.

La función metacont() del paquete meta, nos permite ajustar modelos de meta-análisis para diferencia de medias:

{r} #| eval: false mod_mean <- metacont(n.e = expuestos, # individuos expuestos mean.e = media_exp, # media en expuestos sd.e = sd_exp, # desvío estándar en expuestos n.c = no_exp, # individuos no expuestos mean.c = media_no_exp, # media en no expuestos sd.c = sd_no_exp, # desvío estándar en no expuestos studlab = titulo, # identificador del estudio data = datos) # base de datos de interés}

Odds-ratio

La función metabin() del paquete meta, nos permite ajustar modelos de meta-análisis para el odds-ratio:

{r} #| eval: false mod_or <- metabin(event.e = eventos_exp, # eventos en expuestos n.e = n_exp, # individuos expuestos event.c = eventos_no_exp, # eventos en no expuestos n.c = n_no_exp, # individuos no expuestos sm = "OR", # Odds-ratio studlab = titulo, # identificador del estudio data = datos) # base de datos de interés}

Riesgo relativo

La función metabin() del paquete meta, nos permite ajustar modelos de meta-análisis para el riesgo relativo:

{r} #| eval: false mod_rr <- metabin(event.e = eventos_exp, # eventos en expuestos n.e = n_exp, # individuos expuestos event.c = eventos_no_exp, # eventos en no expuestos n.c = n_no_exp, # individuos no expuestos sm = "RR", # Riesgo relativo studlab = titulo, # identificador del estudio data = datos) # base de datos de interés}

Tasa de incidencia

La función metainc() del paquete meta, nos permite ajustar modelos de meta-análisis para la tasa de incidencia:

{r} #| eval: false mod_IRR <- metainc(event.e = eventos_exp, # eventos en expuestos time.e = tiempo_exp, # tiempo-persona expuestos event.c = eventos_no_exp, # eventos en no expuestos time.c = tiempo_no_exp, # tiempo-persona no expuestos studlab = titulo, # identificador del estudio data = datos) # base de datos de interés}
