Meta-análisis de estudios descriptivos

En epidemiología, las medidas de asociación más utilizadas en estudios descriptivos incluyen la prevalencia, la tasa de incidencia y la correlación. A continuación, se describe cómo ajustar modelos de meta-análisis para cada una de ellas usando el paquete meta. Quienes tengan interés en ampliar el desarrollo matemático de cada caso, pueden consultar los Capítulos 3 y 4.2 de Harrer et al. (2021).

Antes de comenzar con los ejemplos, cargaremos los paquetes necesarios:

# Cargar paquetes
library(meta)

Prevalencia

La prevalencia de un evento de salud se define como la proporción de individuos que presentan el evento \((k)\) dentro de una población o muestra determinada \((n)\):

\[ p = \frac{k}{n} \]

El error estándar de una proporción se estima usando la fórmula:

\[ SE_p = \sqrt{\frac{p(1-p)}{n}} \]

Dado que las proporciones toman valores entre 0 y 1, los valores cercanos a estos extremos pueden generar problemas de asimetría y compresión del error estándar, llevando a sobrestimar la precisión de las estimaciones. Para corregir este efecto, se puede aplicar transformación logit a las proporciones crudas antes de realizar el análisis.

La función metaprop() del paquete meta permite ajustar modelos para prevalencia e incorpora automáticamente la transformación logit con el argumento sm = "PLOGIT". A modo de ejemplo, ajustaremos un modelo utilizando la base de datos dat.crisafulli2020, que contiene datos de 26 estudios sobre la prevalencia de la distrofia muscular de Duchenne en recién nacidos:

# Carga de datos
datos <- dat.crisafulli2020

# Modelo de meta-análisis para prevalencia
mod_prev <- metaprop(
  event = cases,     # Número de individuos con el evento
  n = total,         # Tamaño de la muestra en cada estudio
  studlab = study,   # Identificador único de cada estudio
  data = datos,      # Conjunto de datos
  sm = "PLOGIT",     # Transformación logit para proporciones
  common = FALSE,    # Omitir modelo de efectos fijos
  random = TRUE,     # Ajustar modelo de efectos aleatorios
  backtransf = TRUE, # Convertir resultados a la escala original
  pscale = 100       # Expresar prevalencias como porcentajes
)

En este caso, la prevalencia global estimada es muy baja \((0,02\%, 95\% IC: 0,21-0,24\%)\). Para mejorar la interpretación de estos valores, podemos expresarlos en casos por 10 000 o 100 000 habitantes ajustando el argumento pscale, ya sea en el ajuste del modelo como en la visualización del forest plot:

# Forest plot con casos cada 100 000 habitantes
forest(
  mod_prev, 
  smlab = "Prevalencia (por 100 000 hab.)",
  col.diamond = "#B200B2", # Magenta
  col.square = "#FFFF66", # Amarillo claro
  pscale = 100000
)

Tasa de incidencia

La tasa de incidencia se utiliza cuando el evento de interés es la incidencia de un evento de salud a lo largo del tiempo:

\[ IR = \frac {k}{T} \]​

donde \(k\) es el número de eventos observados y \(T\) es la suma del tiempo-persona en riesgo en cada estudio.

La función metarate() permite ajustar modelos de meta-análisis para tasas de incidencia, aplicando una transformación logarítmica a los datos mediante el argumento sm = "IRLN". Para ejemplificar, usaremos la base de datos dat.nielweise2008 que contiene resultados de 9 estudios sobre incidencia de infecciones sanguíneas relacionadas al uso de cateteres:

# Carga datos
datos <- dat.nielweise2008

# Modelo de meta-análisis para tasas de incidencia
mod_ir <- metarate(
  event = x2i,         # Número de eventos observados
  time = t2i, # Tiempo-persona en riesgo
  studlab = study,    # Identificador único de cada estudio
  data = datos,      # Conjunto de datos
  sm = "IRLN",       # Transformación logaritmo natural para tasas
  common = FALSE,    # Omitir modelo de efectos fijos
  random = TRUE,     # Ajustar modelo de efectos aleatorios
  irscale = 1,     # Tasas por años-persona
  irunit = "años-persona" # Etiqueta para la unidad de tiempo-persona
)

Correlaciones

Las correlaciones indican la fuerza y dirección de la covarianza entre dos variables numéricas continuas y se expresa como:

\[ r_{xy} = \frac{Cov_{xy}}{S_xS_y} \]

donde:

  • \(Cov_{xy}\) es la covarianza entre las variables X e Y.

  • \(S_x\) y \(S_y\) son los desvíos estándar para cada variable.

Como las correlaciones toman valores de entre -1 y 1, se pueden introducir sesgos cuando calculamos el error estándar para muestras pequeñas. Para evitar este sesgo, los datos se transforman usando el coeficiente z de Fisher, que aproxima la muestra a una distribución normal.

La función metacor() permite ajustar modelos de meta-análisis para correlaciones, aplicando la transformación z de Fisher mediante el argumento sm = "ZCOR". Para ejemplificar, usaremos la base de datos dat.aloe2013 que contiene resultados de 5 estudios sobre correlación entre condiciones laborales y salud mental en asistentes sociales:

# Carga datos
datos <- dat.aloe2013

# Modelo de meta-análisis para tasas de incidencia
mod_cor <- metacor(
  cor = R2,       # Número de eventos observados
  n = n,            # Tamaño de la muestra
  studlab = study,  # Identificador único de cada estudio
  data = datos,     # Conjunto de datos
  sm = "ZCOR",      # Transformación z de Fisher
  common = FALSE,   # Omitir modelo de efectos fijos
  random = TRUE,    # Ajustar modelo de efectos aleatorios
  )

# Resumen del modelo ajustado
